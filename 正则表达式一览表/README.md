# 正则表达式一览

### 校验数字的表达式

- 数字： 						^\d+$
- n位的数字： 					^\d{n}$
- 至少n位的数字：					^\d{n,}$
- m-n位的数字：					^\d{m,n}$
- 零和非零开头的数字：				^(0|[1-9]\d*)$
- 非零开头的最多带两位小数的数字：	^[1-9]\d*(\.\d{1,2})?$
- 带1-2位小数的正数或负数：		^(\-)?\d*\.\d{1,2}$
- 正数、负数、和小数：				
- 有两位小数的正实数：				^\d+\.\d{2}$
- 有1~3位小数的正实数：			^\d+\.\d{1,3}$
- 非零的正整数：					^[1-9]\d*$
- 非零的负整数：					^-[1-9]\d*$
- 非负整数：						^\d+$或者^([1-9]\d*|0)$
- 非正整数：						^(-[1-9]\d*|0)$或者^(-\d+|0)$
